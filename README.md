# Tox21 多任务毒性预测（基于 GINE 图神经网络）

本项目使用图同构网络（GINE，Graph Isomorphism Network with Edge Features）对 Tox21 数据集的 12 个毒性终点进行多标签分类预测。与传统分子描述符 + DNN 方法不同，本模型直接以分子图为输入，自动学习原子和键的高阶特征，在药物早期毒性筛选中具有更好的泛化能力。

毒性预测是药物研发中的关键环节：

- 高毒性化合物可能在临床阶段失败，导致巨大损失；
- 早期通过计算模型筛选毒性风险，可大幅降低实验成本；
- 多任务学习可同时预测多种毒性终点，提供全面的安全性评估。

本项目基于 Tox21 数据集（约 8000 个化合物，12 种毒性实验），通过分子的 SMILES 字符串构建分子图，训练一个 GINE 图神经网络来预测每个化合物在各个毒性任务上的风险概率。

## 1. 数据处理

**输入数据**：`tox21.csv`，包含以下关键列：
- `smiles`：分子的 SMILES 字符串表示。
- `mol_id`：分子标识符（可选）。
- 12 个毒性任务列（如 `NR-AR`, `NR-ER`, `SR-p53` 等）：每个任务取值为 0（无毒）、1（有毒）或缺失（-1）。

**数据划分**：
- 采用 **Murcko 骨架（Scaffold）** 划分，避免相同骨架的分子分散在不同集合中。
- 划分比例：训练集:验证集:测试集 = 8:1:1。

**分子图构建**：
- 将每个 SMILES 转换为无向图，原子为节点，键为边。
- **节点特征（15 维）**：
  - 原子序数（归一化到 [0,1]）
  - 度数（归一化）
  - 形式电荷
  - 连接的氢原子数（归一化）
  - 隐式价（归一化）
  - 杂化类型（整数编码）
  - 是否芳香碳
  - 是否成环
  - 最小环大小（归一化）
  - 是否在芳香环中
  - 手性标签
  - Gasteiger 部分电荷（归一化）
  - 是否为 sp 杂化
  - 是否为 sp² 杂化
  - 是否为 sp³ 杂化

- **边特征（3 维）**：
  - 键类型（单键=1.0，双键=2.0，三键=3.0，芳香键=1.5）
  - 是否为芳香键
  - 是否在环中

## 2. 模型结构（GINE）

GINE（Graph Isomorphism Network with Edge Features）在经典 GIN 基础上引入了边特征，能够更精细地建模键的差异。模型结构如下：
输入节点特征(15) → Linear(64) →
[
GINEConv(MLP) → BatchNorm → ReLU → Dropout(0.5)
] × 3 →
全局均值池化 → Dropout(0.5) → Linear(64 → 12) → 输出 logits

具体说明：
- **MLP 结构**（每个 GINEConv 内部）：`Linear(64,64) → BN → ReLU → Dropout → Linear(64,64) → BN → ReLU → Dropout`
- **GINEConv 层数**：3 层，每层之后接批归一化和 ReLU 激活。
- **输出层**：输出 12 维 logits，对应 12 个任务的未归一化分数。
- **激活函数**：训练时使用 BCEWithLogitsLoss（内置 sigmoid）；预测时通过 `torch.sigmoid` 转换为概率。

## 3. 训练与评估

### 不平衡处理
由于有毒样本通常远少于无毒样本，采用双重策略：
- **加权采样器**：训练集中，如果一个分子至少在某个任务上为 1（有毒），其采样权重增加 `OVER_SAMPLE_RATIO`（默认 2.0），使模型在每个 batch 中看到更多正样本。
- **正样本权重**：计算每个任务的 `pos_weight = 负样本数 / 正样本数`，传入 `BCEWithLogitsLoss` 中，增大正类的误分类代价。

### 损失函数
- 带掩码的二元交叉熵：忽略标签为 -1 的样本。
- 多任务总损失 = 各任务有效样本的平均损失之和。

### 优化器与调度
- **优化器**：AdamW（学习率 0.0001，权重衰减 5e-4）。
- **学习率调度**：ReduceLROnPlateau（监控验证集平均 AUC，模式 `max`，patience=3，因子 0.5）。
- **梯度裁剪**：最大梯度范数 0.5。

### 早停
- 监控验证集平均 AUC，连续 **10 个 epoch** 不提升则停止训练，并恢复最佳模型权重。

### 评估指标
- 每个任务单独计算：
  - AUC（ROC 曲线下面积）
  - 准确率（Accuracy）
  - 精确率（Precision）
  - 召回率（Recall）
  - F1 分数
  - 混淆矩阵（TP, FP, TN, FN）
- 综合指标：12 个任务的 **平均 AUC**。
